> * Clone this repo, and edit in your editor:
> 
>   ```
>   git clone https://github.com/artsy/guides.git
>   cd guides
>   yarn install
>   code .
>   ```
> 
>   This will set up prettier to format your markdown and inline TS/CSS/GraphQL.
